## asciinema工具使用
#### 按照官网上的方法安装配置asciinema
```bash
#install
sudo apt-get install asciinema
#link account
asciinema auth
```
#### asciinema使用方法
```bash
#start recording
asciinema rec
 ......
#stop recording
exit
```
